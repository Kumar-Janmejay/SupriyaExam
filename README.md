# AEM Assets Expert Exam Simulator

A Streamlit-based exam simulator application for testing knowledge on AEM Assets.

## Features
- 🎯 Interactive exam questions
- ⏱️ Real-time timer
- 📊 Comprehensive analytics and results
- 📱 Mobile-friendly interface

## Local Setup

1. Clone the repository
2. Install dependencies:
   ```
   pip install -r requirements.txt
   ```
3. Run the app:
   ```
   streamlit run app.py
   ```

## Deployment

This app is deployed on Streamlit Cloud. Visit the live app using the provided link.

## Technologies
- Streamlit
- Python
